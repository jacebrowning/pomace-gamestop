# Setup

```
brew install pipx

git clone https://github.com/jacebrowning/pomace-gamestop
cd pomace-gamestop
```

# Usage

```
pipx run pomace run https://www.gamestop.com/video-games/xbox-one/games/products/mlb-the-show-21/11116256.html
```
